[Brief overview of the game and its premise]

## Important Notes

1. **Cover Images**: ALWAYS generate and include a cover image at the top of each review
2. **Metadata**: Ensure all metadata fields are filled accurately
3. **Platform**: Specify the platform (Steam, Pirated, GOG, etc.)
4. **Achievements**: Log achievement progress when available
5. **Playtime**: Use the format `Xh Ym` (e.g., `29h 15m`)
6. **Rating Scale**: Use the 6-star scale (★☆☆☆☆☆ to ★★★★★★) consistent with other Game-Reviews
7. **Update Files**: Always update README.md and temporal_distribution.md when adding new games

## Cover Image Generation

When generating cover images, use prompts that:
- Match the game's art style and era
- Include the game title
- Reflect the genre and tone (dark for thrillers, vibrant for action, etc.)
- Feature key imagery or characters from the game
- Feel like an official box art or promotional cover
